# Shot-Chart-Generator

Simple Python Tkinter app for creating basketball shot charts with make and miss buttons. 

* Allows loading any court image with a file picker
* Make - Green cirle
* Miss - Red X
* Exports annotated chart as a PNG image

# Requirements

- Python 3.7+
- Pillow library
